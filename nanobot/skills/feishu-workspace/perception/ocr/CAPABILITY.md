# OCR Perception

这是 `feishu-workspace` 下的图片与附件感知能力，用于读取当前飞书消息里的图片文字。

## 何时使用

- OCR / 识别图片文字 / 提取截图文字
- 提取营业执照信息 / 提取企业信息 / 识别证照信息

## 基本规则

1. 先检查当前用户消息是否包含图片。
2. 如果没有图片，停止并请用户重新发送图片。
3. 如果有多张图片，只处理第一张，并让用户按张继续发。
4. 默认使用通用 OCR；只有在用户明确要求企业/证照字段提取时，才使用企业 OCR 模式。

## 边界

- 不要编造看不清的文字。
- 只提取图片中可见的信息。
- 不要联网核验企业真伪。
- OCR-only 任务不要额外调用工具。
