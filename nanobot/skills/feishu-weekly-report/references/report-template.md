# 周报模板约束

在完成字段映射并筛选出本周记录后，按下面的结构组织文档。

## 文档结构

1. 一级标题：`{周报标题}`
2. 段落：统计时间范围、数据来源
3. 二级标题：`本周概览`
4. 段落或无序列表：本周总体情况
5. 二级标题：`重点进展`
6. 无序或有序列表：重要事项与进展
7. 二级标题：`风险/阻塞`
8. 引用或无序列表：风险、卡点、待决策项
9. 二级标题：`下周计划`
10. 有序列表：下周动作
11. 二级标题：`附录/原始条目摘要`
12. 无序列表：原始条目摘要，必要时按事项列出

## block 约束

只使用这些 block 类型：

- 标题 1 / 2 / 3
- 文本段落
- 无序列表
- 有序列表
- 引用

不要使用：

- 嵌套 children
- 表格
- 图片
- 代码块
- 任意位置插入

## 内容压缩规则

- `本周概览`：优先总结数量、完成度、主要主题，不要直接抄所有记录。
- `重点进展`：优先写对外可汇报的结果。
- `风险/阻塞`：如果没有，就明确写“本周未识别到显著风险/阻塞”。
- `下周计划`：只写可执行动作，避免空话。
- `附录/原始条目摘要`：控制为摘要，不粘贴整表。
